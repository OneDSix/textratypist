# LML Extension

This addon adds TextraTypist support for [LibGDX Markup Language](https://github.com/crashinvaders/gdx-lml/tree/master/lml)

```groovy
implementation "com.github.tommyettinger:textratypist:LMLExtension:0.10.0"
```
